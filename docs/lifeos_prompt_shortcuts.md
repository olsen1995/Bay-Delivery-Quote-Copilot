LifeOS Prompt Shortcuts  
Fast Ways to Get the Right Help

---

## PURPOSE

Prompt shortcuts allow the user to quickly steer LifeOS  
without needing to explain tone, depth, or intent.

These are optional and never required.

---

## GENERAL SHORTCUTS

- **"Step-by-step:"**  
  → Provide clear, numbered instructions  

- **"Quick version:"**  
  → Short, efficient answer with minimal explanation  

- **"Explain like I’m learning:"**  
  → Teaching-focused, patient explanation  

- **"Sanity check:"**  
  → Verify if something makes sense or is set up correctly  

---

## DAILY LIFE SHORTCUTS

- **"Low energy day:"**  
  → Favor easy wins and minimal steps  

- **"I’m stuck:"**  
  → Trigger daily flow guidance  

- **"What’s the simplest way:"**  
  → Prioritize reliability over optimization  

---

## HOUSEHOLD & DIY SHORTCUTS

- **"Can I DIY this?"**  
  → Assess difficulty, safety, and alternatives  

- **"Fix-it:"**  
  → Troubleshooting and repair steps  

- **"Cleaning mode:"**  
  → Surface-safe cleaning guidance  

- **"Laundry help:"**  
  → Fabric-safe washing and drying advice  

---

## COOKING SHORTCUTS

- **"What can I make with this?"**  
  → Ingredient-based recipe suggestions  

- **"Simple meal:"**  
  → Minimal prep, minimal cleanup  

- **"Texture matters:"**  
  → Focus on doneness and mouthfeel  

---

## TECH & DEVICE SHORTCUTS

- **"Optimize safely:"**  
  → Reversible, low-risk settings only  

- **"Is this setting okay?"**  
  → Explanation before recommendation  

- **"Fix my device:"**  
  → Troubleshooting without risky changes  

---

## USAGE NOTE

Prompt shortcuts:  
- Are optional  
- Can be mixed with normal questions  
- Do not change LifeOS core behavior  
- Exist to reduce friction
